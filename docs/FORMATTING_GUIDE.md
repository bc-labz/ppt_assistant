# PPT 排版優化指南

## 優化內容總覽

本次更新大幅改善了 PPT 的文字對齊和排版效果,提供更專業的視覺呈現。

## 主要改進

### 1. 標題頁優化
- **標題**: 44pt 粗體,居中對齊
- **副標題**: 28pt,居中對齊
- 清晰的視覺層次

### 2. 內容頁優化
- **頁面標題**: 32pt 粗體,左對齊
- **正文內容**: 20pt,左對齊
- **列表項目**: 每項間距 6pt,提高可讀性
- 自動換行處理

### 3. 雙欄佈局優化
- **標題**: 32pt 粗體,左對齊
- **左右欄**: 18pt,左對齊
- 欄位寬度平均分配 (各 6 英寸)
- 適當的段落間距 (12pt)
- 啟用自動換行

### 4. 圖片頁優化
- **標題**: 32pt 粗體,左對齊
- **圖片**: 居中放置,寬度 9.333 英寸
- **圖片說明**: 16pt 斜體,居中對齊

### 5. 表格頁優化
- **標題**: 32pt 粗體,左對齊
- **表頭**:
  - 18pt 粗體,居中對齊
  - 藍色背景 (RGB: 68, 114, 196)
  - 白色文字
- **數據單元格**:
  - 16pt,居中對齊
  - 交替行背景色 (淺藍色: RGB 217, 226, 243)
- 表格寬度: 11.333 英寸

### 6. 圖表頁優化
- **標題**: 32pt 粗體,左對齊
- **圖表尺寸**: 10.333 x 5 英寸
- **圖例**: 顯示在右側
- 清晰的數據視覺化

## 字體大小規範

| 元素 | 字體大小 | 粗體 | 對齊方式 |
|-----|---------|------|---------|
| 主標題 | 44pt | 是 | 居中 |
| 副標題 | 28pt | 否 | 居中 |
| 頁面標題 | 32pt | 是 | 左對齊 |
| 正文內容 | 20pt | 否 | 左對齊 |
| 雙欄文字 | 18pt | 否 | 左對齊 |
| 表格表頭 | 18pt | 是 | 居中 |
| 表格數據 | 16pt | 否 | 居中 |
| 圖片說明 | 16pt | 否 (斜體) | 居中 |

## 間距規範

- **段落前間距**: 0pt (避免不必要的空白)
- **段落後間距**: 12pt (雙欄佈局)
- **列表項間距**: 6pt (內容頁)
- **標題與內容間距**: 自動調整

## 顏色方案

### 表格配色
- **表頭背景**: RGB(68, 114, 196) - 專業藍色
- **表頭文字**: RGB(255, 255, 255) - 白色
- **交替行背景**: RGB(217, 226, 243) - 淺藍色
- **數據文字**: 黑色 (默認)

## 佈局尺寸

### 標準元素位置
- **標題位置**: 左 0.5", 上 0.3", 寬 12.333"
- **雙欄左欄**: 左 0.5", 上 1.5", 寬 6", 高 5.5"
- **雙欄右欄**: 左 6.833", 上 1.5", 寬 6", 高 5.5"
- **圖片**: 左 2", 上 1.8", 寬 9.333"
- **表格**: 左 1", 上 1.8", 寬 11.333", 高 5"
- **圖表**: 左 1.5", 上 1.8", 寬 10.333", 高 5"

## 最佳實踐

### 文字內容
1. 使用簡潔的標題 (不超過 10 個字)
2. 列表項目控制在 5-7 項
3. 每行文字不超過 60 個字符
4. 避免過長的段落

### 表格設計
1. 表頭使用粗體和對比色
2. 數據居中對齊,便於閱讀
3. 使用交替行背景色提高可讀性
4. 控制欄位數量 (建議不超過 6 欄)

### 圖表設計
1. 選擇合適的圖表類型
2. 確保圖例清晰可見
3. 數據系列不超過 5 個
4. 使用對比明顯的顏色

### 圖片使用
1. 使用高解析度圖片
2. 保持圖片比例
3. 添加簡短的說明文字
4. 確保圖片與主題相關

## 自定義調整

如需調整默認設置,可以修改 `ppt_assistant.py` 中的以下參數:

```python
# 字體大小
p.font.size = Pt(20)  # 調整數值

# 對齊方式
p.alignment = PP_PARAGRAPH_ALIGNMENT.LEFT  # LEFT, CENTER, RIGHT

# 段落間距
p.space_before = Pt(6)
p.space_after = Pt(12)

# 顏色設置
cell.fill.fore_color.rgb = RGBColor(68, 114, 196)
```

## 技術細節

### 文字框自動換行
所有文字框都啟用了自動換行功能:
```python
text_frame.word_wrap = True
```

### 段落格式化
統一的段落格式化方法確保一致性:
```python
def _format_text_frame(self, text_frame, font_size=18, align=PP_PARAGRAPH_ALIGNMENT.LEFT):
    for paragraph in text_frame.paragraphs:
        paragraph.alignment = align
        paragraph.font.size = Pt(font_size)
        paragraph.space_before = Pt(0)
        paragraph.space_after = Pt(12)
```
