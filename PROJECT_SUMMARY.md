# PPT 助手工具 - 項目總結

## 項目概述

本項目實現了一個功能完整的 Python PPT 生成工具,能夠從 JSON 格式的配置文件讀取內容,套用 .pptx 模板,並輸出包含豐富元素的專業簡報。

## 核心功能

### 1. JSON 格式輸入
- 使用結構化的 JSON 格式定義簡報內容
- 支持靈活的配置選項
- 易於程式化生成和管理

### 2. 模板支持
- 可以使用現有的 .pptx 文件作為模板
- 自動繼承模板的樣式和主題
- 如果沒有模板,自動創建新簡報

### 3. 多種佈局類型

| 佈局類型 | 說明 | 適用場景 |
|---------|------|---------|
| title | 標題頁 | 簡報首頁 |
| content | 內容頁 | 文字說明、要點列表 |
| two_column | 雙欄佈局 | 對比、並列內容 |
| image | 圖片頁 | 展示圖片和說明 |
| table | 表格頁 | 數據展示 |
| chart | 圖表頁 | 數據可視化 |

### 4. 圖表支持

支持四種常用圖表類型:
- **柱狀圖** (column): 垂直柱狀圖,適合比較數據
- **條形圖** (bar): 水平條形圖,適合排名展示
- **折線圖** (line): 趨勢分析
- **圓餅圖** (pie): 比例展示

### 5. 豐富的內容元素

- 文字內容(支持單段或列表)
- 圖片插入(支持多種格式)
- 表格生成(自動格式化)
- 圖表生成(多系列數據支持)

## 技術實現

### 核心依賴
- **python-pptx**: PowerPoint 文件操作庫
- **Python 3.11**: 運行環境

### 代碼結構

```python
class PPTAssistant:
    - __init__(): 初始化,載入配置和模板
    - create_presentation(): 主流程,遍歷所有投影片
    - _add_title_slide(): 添加標題頁
    - _add_content_slide(): 添加內容頁
    - _add_two_column_slide(): 添加雙欄頁
    - _add_image_slide(): 添加圖片頁
    - _add_table_slide(): 添加表格頁
    - _add_chart_slide(): 添加圖表頁
```

### 設計特點

1. **模組化設計**: 每種佈局類型都有獨立的處理方法
2. **錯誤處理**: 對不存在的文件和錯誤配置有適當處理
3. **靈活性**: 支持模板和無模板兩種模式
4. **可擴展性**: 易於添加新的佈局類型和功能

## 使用示例

### 最簡單的使用

```bash
python3 ppt_assistant.py example_config.json
```

### 程式化使用

```python
from ppt_assistant import PPTAssistant

assistant = PPTAssistant('config.json')
assistant.create_presentation()
```
